# Blackjack

Blackjack game written in C++.


## Features

* Player vs dealer
* Standard blackjack rules
* Ace can count as 1 or 11
* Dealer hits until 17
* 8 Deck shoe reshuffled after every hand
* Player can double down


## Build

Compile with any C++ compiler (C++11 or newer).

Example using g++:

g++ *.cpp -o blackjack

## Run

./blackjack
